Design Rationale

Refines the existing wave-bridge concept with the correct brand gradient (#5E72E4 → #825EE4), aligning to the source wave-bridge.svg. Neural nodes are added at the three pillar-tops to reinforce the AI identity without introducing a brand redesign.

  • Minimal change — stakeholders recognize the existing brand symbol
  • Gradient fix resolves the inconsistency between SVG source and background XML
  • Glowing nodes visually communicate AI/neural processing
  • Notification icon (single wave arc + 3 dots) is legible even at 16dp
  • Lowest implementation cost — update gradient stops + add 3 circle elements
